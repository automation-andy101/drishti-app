# Drishti 🧘

A yoga sequencing app for teachers and students.

Built with Next.js, Supabase, Prisma, Tailwind, and shadcn/ui.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Database:** Supabase (PostgreSQL)
- **ORM:** Prisma
- **Auth:** Supabase Auth
- **Styling:** Tailwind CSS + shadcn/ui
- **State:** Zustand

## Getting Started

1. Clone the repo
2. Copy `.env.local.example` to `.env.local` and fill in your Supabase credentials
3. Run `npm install`
4. Run `npm run dev`
